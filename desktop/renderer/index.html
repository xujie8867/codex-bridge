计费模式

混合模式适合 GPT 走 Codex 订阅、其他模型走各自 API；全部 API 模式适合统一走中转或团队接口。

运行控制

更新会生成模型目录并写入 CodexBridge 配置；Router 开关负责本地网关是否接收 Codex 请求。

常规流程：选模型、填密钥、更新配置、启动 Router，然后重启 Codex。右上角“初始化 Codex 配置”用于恢复到 CodexBridge 写入前的备份。
